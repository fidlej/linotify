Run linotify-agent.tar.gz on your server to send the stats.
Use SERVER_KEY = '${server.get_secret_key()}' in its libagent/config.py.

Agent usage

Prepare it: 

wget http://www.linotify.com/static/download/linotify-agent.tar.gz
tar xzf linotify-agent.tar.gz
cd linotify-agent
sed -i -e 's/^SERVER_KEY.*/SERVER_KEY = "${server.get_secret_key()}"/' libagent/config.py
./agent.py -v

Schedule it by cron to run every minute: 

(crontab -l ; echo "* * * * * `pwd`/agent.py") | crontab -


The received data will be then visible on server graphs.
